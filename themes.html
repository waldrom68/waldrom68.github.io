  • Desarrollado sólo con CSS3, JS y HTML.

    • Características de diseño y codificación:

  • Modo responsivo y Modo Dark/Light.
  • Icons-SVG: algunos usando la etiqueta <image> de HTML y otros incluyéndolos dentro del body como un <symbol> para incrustar el código vectorial, e invocarlos usando su Id.
  • Estilo especifico para las salidas de impresora con @media print {...}.
  • Fuentes: también se emplearon dos metodologías, una mediante @import url(....) y la otra con @font-face {....}.
  • Gracias a w3s no necesité dedicar mucho tiempo para el navBar, ni para el diseño de este "Acerca de".
